Reglas aplicables a los atributos

  • El atributo tiene que aparecer despues del nombre la etiqueta HTML.
  • El nombre del atributo tiene que ir seguido de un signo (=).
  • El valor de atributo tiene que ir inmediatamente despues del signo igual
  • El valor del atributo siempre tiene que ir entre comillas simples o dobles.

Existen unos 147 nombres de colores (atributos para color) que podemos encontrar en sitios como: www.w3schools.com/html/html_colornames.asp

Para valores hexdecimales se mezclan colores especificando cantidades definidas de colores primarios: rojo, verde, azul en el formato hexadecimal "#RRGGBB".
Ejemplo: #FF00FF (Morado: Rojo y azul).
Valores RGB en formato decimal: "RGB(R,G,B)".En este caso los valores se especifican con números del 0 al 255 o con porcentajes
Ejemplo:"rgb(255,0,255)" (Maximo rojo, cero verde, maxino azul = Morado)

Atributos comunes:

Atributo id:
El atributo id aisgna un identificador único a una etiqueta. Muy útil cuando referenciamos etiquetas desde scripts.
Atributo class:
El atributo clase asisgna el mismo identificador a más de una etiqueta. Muy útil cuando referenciamos etiquetas desde css para darle estilos al contenido.
Se pueden aplicar estilos directamente sobre las etiquetas usando el atributo: style="style-definition; style-definition"
