Reglas de CSS

Las reglas de CSS se componen de selectores. un selector es una forma de referirnos a una parte del documento HTML.

Ejemplos:
Referencias a etiquetas (Selector de etiquetas) 
body{

      background-color: coral;
}

En CSS hay muchos estilos que se heredan con los mismos valores (desde la etiqueta del selector hacia abajo)

Para importar una hoja de estilo dentro de otra hoja de estilo:
@import './botones.css'

Selectores de clase

El atributo clase (class) se puede repetir, pero el atributo id es único en cada página.

otros valores de propiedades:
inline-size:32; (Ancho del texto de la linea)
block-size: 32; (Alto del texto en el eje Y)
background-color: transparent (Color de fondo del elemento)

Cuando el ancho es más grande los botones (por ejemplo) se empiezan a repetir, pero esto se puede corregir con background-repeat: no-repeat
background-position: center (Centra el contenido si la caja es más grande)

Una pseudoclase CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado. Por ejemplo :hover (aplicará un estilo cuando el usuario haga hover sobre el elemento especificado por el selector)
