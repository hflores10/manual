Reglas de CSS

Las reglas de CSS se componen de selectores. un selector es una forma de referirnos a una parte del documento HTML.

Ejemplos:
Referencias a etiquetas (Selector de etiquetas) 
body{

      background-color: coral;
}

En CSS hay muchos estilos que se heredan con los mismos valores (desde la etiqueta del selector hacia abajo)

Para importar una hoja de estilo dentro de otra hoja de estilo:
@import './botones.css'

Selectores de clase

El atributo clase (class) se puede repetir, pero el atributo id es único en cada página.

otros valores de propiedades:
inline-size:32; (Ancho del texto de la linea)
block-size: 32; (Alto del texto en el eje Y)
background-color: transparent (Color de fondo del elemento)

Cuando el ancho es más grande los botones (por ejemplo) se empiezan a repetir, pero esto se puede corregir con background-repeat: no-repeat
background-position: center (Centra el contenido si la caja es más grande)

Una PSEUDOCLASE CSS es una palabra clave que se añade a los selectores y que especifica un estado especial del elemento seleccionado. Por ejemplo :hover (aplicará un estilo cuando el usuario haga hover (flotar/El mouse este sobre en elemento) sobre el elemento especificado por el selector).

Display Flex

display:flex
Debes tener un container principal al que agregar flex y ese container va a heredar a los hijos y a los hijos de sus hijos.
Si el padre tiene display flex los hijos se acomodan de forma flexible. (Un padre que emcapsula a todos los hijos). display-direction:row; (Dirección en la que se van a acomodar los hijos).
flex-direction:row-reverse; (Invierte la posición normal de los conte nedores). flex-direction:colum-reverse; (Invierte la posición de los elementos dentro de la columna).
flex-wrap:wrap; (Cada uno de los elementos caundo ya no tiene espacio se bajan a la siguiente fila de forma automática). flex-wrap:wrap-reverse (Igual que el anterior pero invierte las posici ciones).
justify-content: center Nos ayuda a alinear el contenido de forma horizontal, centrar las cosas. justify-content:flex-end (todo a la derecha). :flex-start (todo a la izqyuerda) : spacearound (Espacio entre contenedores) :space-evenly (Espacio justificado)

Posicionamiento de forma vertical

align-items:center; (Alineación al centro).
align-items:flex-end (Alineado hasta la parte de abajo de padre).
align-items:flex-start (Alineación arriba).
align-items:flex-stretch (Altura del contenedor padre).
align-items:flex-baseline (Toma el tamaño del contenido).

Orden de los contenedores

flex-grow:1; (Asignarlo al contenedor que deseas que cresca).

flex-wrap; (En contenedor principal).
flex-basis:10rem; (En contenedores hijos).
flex-grow:1; (En contenedores hijos).
